## 📖 Описание

**PerlinForge** — инструмент для массовой генерации случайных процедурных текстур с полным контролем над параметрами. Создаёт датасеты изображений, пригодные для обучения нейросетей, аугментации данных, тестирования пайплайнов обработки изображений или использования в качестве фонов и заполнителей.

Каждое изображение формируется послойным наложением:

```
┌─────────────────────────────────┐
│  Слой 1 · Облака (Perlin fBm)   │  ← фрактальный броуновский шум
├─────────────────────────────────┤
│  Слой 2 · Цветной RGB-шум       │  ← гауссово сглаживание
├─────────────────────────────────┤
│  Слой 3 · Монохромное зерно     │  ← имитация плёночного grain
├─────────────────────────────────┤
│  Пост · JPEG-деградация (опц.)  │  ← многопроходное пересжатие
└─────────────────────────────────┘
```

## ✨ Возможности

- 🎨 **Градиентные карты** — колоризация любого слоя произвольным градиентом с автосортировкой по яркости (luma)
- 🔊 **Фрактальный шум Перлина** — настраиваемые октавы, persistence, lacunarity, базовое разрешение
- 📐 **Случайные разрешения** — автоматический выбор из стандартных тиров (0.5K–4K) и соотношений сторон (1:1, 3:4, 9:16, 16:9 и др.)
- 💥 **Симуляция JPEG-артефактов** — 1–4 прохода сжатия с качеством 10–40 для имитации реальной деградации
- ⚡ **Параллельная генерация** — `ProcessPoolExecutor` с автоматическим определением числа ядер
- 📋 **Полные метаданные** — JSON-файл со всеми параметрами каждого изображения для воспроизводимости
- 🎲 **Детерминистичность** — фиксация `seed` для точного воспроизведения любого изображения

## 🚀 Быстрый старт

### Установка зависимостей

```bash
uv add numpy scipy pillow tqdm
```

### Генерация датасета (1000 изображений)

```bash
uv run perlinforge.py
```

### Программное использование

```python
from perlinforge import generate_random_image, create_ml_dataset

# Одно изображение с заданными параметрами
img = generate_random_image(
    width=1920,
    height=1080,
    cloud_octaves=8,
    cloud_persistence=0.6,
    seed=42,
)
img.save("texture.png")

# Датасет из 500 изображений в 8 процессов
create_ml_dataset(
    num_images=500,
    output_dir="my_dataset",
    max_workers=8,
)
```

## ⚙️ Параметры генерации

### `generate_random_image`

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `width`, `height` | `int` | — | Размеры изображения |
| `enable_clouds` | `bool` | `True` | Слой облаков (Perlin fBm) |
| `enable_rgb_noise` | `bool` | `True` | Слой цветного шума |
| `enable_grain` | `bool` | `True` | Слой монохромного зерна |
| `cloud_gradient` | `GradientMap \| None` | `None` | Градиент для облаков |
| `grain_gradient` | `GradientMap \| None` | `None` | Градиент для зерна |
| `cloud_base_res` | `int` | `4` | Базовое разрешение сетки |
| `cloud_octaves` | `int` | `6` | Количество октав fBm |
| `cloud_persistence` | `float` | `0.5` | Затухание амплитуды |
| `cloud_lacunarity` | `float` | `2.0` | Множитель частоты |
| `rgb_noise_opacity` | `float` | `0.30` | Непрозрачность RGB-шума |
| `grain_opacity` | `float` | `0.20` | Непрозрачность зерна |
| `seed` | `int \| None` | `None` | Seed для воспроизводимости |

### `create_ml_dataset`

| Параметр | Тип | По умолчанию | Описание |
|---|---|---|---|
| `num_images` | `int` | `1000` | Количество изображений |
| `output_dir` | `Path \| str` | `random_noisy_textures` | Директория вывода |
| `max_workers` | `int \| None` | `None` (авто) | Число параллельных процессов |

## 📁 Структура вывода

```
random_noisy_textures/
├── tex_0000.png
├── tex_0001.png
├── tex_0002.jpg      ← с JPEG-деградацией
├── tex_0003.png
├── ...
└── _metadata.json    ← параметры всех изображений
```

## 🧠 Применение

| Задача | Как использовать |
|---|---|
| **Обучение автоэнкодеров** | Подача зашумлённых текстур для обучения денойзингу |
| **Аугментация данных** | Наложение текстур как фоновый шум на реальные изображения |
| **Тестирование JPEG-восстановления** | Пары «чистое / деградированное» благодаря фиксации seed |
| **Генеративные модели** | Тренировочные данные для GAN / диффузионных моделей |
| **Бенчмаркинг пайплайнов** | Контролируемая генерация изображений известных параметров |
